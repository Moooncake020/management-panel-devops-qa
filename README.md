\# Management Panel DevOps \& QA



ASP.NET Core MVC ile geliştirilmiş görev, kullanıcı ve organizasyon yönetim panelidir. Bu repository aynı zamanda uygulamalı DevOps ve QA öğrenme projesi olarak kullanılmaktadır.



\## Projenin Amacı



Yönetici ve çalışanların görevleri, kullanıcıları, departmanları ve organizasyon yapılarını tek bir panel üzerinden yönetebilmesini sağlamak.



\## Temel Özellikler



\- Rol tabanlı kullanıcı yetkilendirmesi

\- Görev oluşturma ve atama

\- Görev durumu ve öncelik yönetimi

\- Kullanıcı yönetimi

\- Departman ve unvan yönetimi

\- Bildirim sistemi

\- Yorum ve aktivite geçmişi

\- Kanban ve takvim görünümleri

\- Dosya ve görsel yükleme



\## Kullanılan Teknolojiler



\- ASP.NET Core MVC

\- C#

\- Entity Framework Core

\- SQL Server

\- Razor Views

\- JavaScript

\- Tailwind CSS

\- Node.js ve npm



\## DevOps ve QA Hedefleri



Bu proje üzerinde aşamalı olarak aşağıdaki yapılar kurulacaktır:



\- Manuel test senaryoları

\- Unit ve integration testleri

\- Playwright E2E testleri

\- GitHub Actions CI pipeline

\- Docker ve Docker Compose

\- Staging ve production deployment

\- Loglama ve monitoring

\- Performans ve güvenlik testleri
